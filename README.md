# My resume

This is a game-like resume built with PixiJs.

![interactive resume](https://media4.giphy.com/media/v1.Y2lkPTc5MGI3NjExZXZ0bTY1Z3JkZzNubW1wbTUzcnU0Zmw3YWw3d2RnM2E4Njdya254cyZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/SRZgIFqL3BxUshGZeN/giphy.gif)

## Quick start

```bash
cd interactive-resume
npm install
npm run dev
```

Then open `http://localhost:5173` in a browser.
